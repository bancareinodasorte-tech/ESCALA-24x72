ESCALA 24x72 - COMO USAR NO CELULAR

IMPORTANTE:
O problema que apareceu no print não é do sistema. O iPhone abriu o arquivo HTML dentro do aplicativo MediaConverter, e esse aplicativo bloqueia os botões.

FORMA CORRETA DE USAR:
1) Publique esta pasta no GitHub Pages, Netlify ou qualquer hospedagem simples.
2) Abra o link pelo Safari no iPhone ou Chrome no Android.
3) No iPhone: toque em Compartilhar > Adicionar à Tela de Início.
4) No Android: Chrome > menu ⋮ > Adicionar à tela inicial.

Depois disso, ele abre como aplicativo.

FUNÇÕES:
- Cadastro de policiais
- Geração automática da escala 24x72
- Turno A e Turno B com titular substituído
- Textos de permuta para copiar
- Salvar dados no celular
- Limpar tudo para outro grupo usar
- Gerar PDF/Imprimir
- Gerar imagem SVG da escala
